(1) Zoom animation


Brief explanation:

animating the image with the class "animate".I am using the "zoom" animation which scales the image up by 20% over a duration of 2 seconds. I am also adding a 1-second delay before the animation starts and using "forwards" for the animation-fill-mode so that the image stays at its final size after the animation finishes.

(2) Fade-in animation


Brief explanation:

defined a div element with the class fade-in, which initially has an opacity of 0 (i.e. it's invisible). Then, I've defined a CSS animation using the @keyframes rule, which gradually increases the opacity of the element from 0 to 1 over a duration of 1 second using an "ease-in" timing function. Finally, I apply the fade-in animation to the element using the animation property, which causes the animation to run when the element is first displayed, and then "freezes" the element in its final state using the forwards keyword
